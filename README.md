# 🌌 Habit Tracker

A full-stack habit tracking application designed to help users build consistency through daily streaks and track progress with a clean, interactive interface.

## ✨ Features

- 🔐 User authentication with secure login/signup (JWT)
- 📝 Create and manage daily habits
- 🔥 Track current and best streaks
- 📊 Analytics dashboard for progress tracking
- 🌌 Constellation-based milestone system
- ⚡ Responsive and modern UI

## 🏗️ Tech Stack

### Frontend
- React.js
- Tailwind CSS

### Backend
- Spring Boot
- Spring Security (JWT)
- Spring Data JPA

### Database
- PostgreSQL (Supabase)

### Deployment
- Frontend → Vercel
- Backend → Render
