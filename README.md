# 🌌 Habit Tracker

A full-stack habit tracking application designed to help users build consistency through daily streaks and track progress with a clean, interactive interface.

---

## 🚀 Live Demo

https://habit-tracker-mu-dusky.vercel.app

---

## ✨ Features

* User authentication with secure login/signup
* Create and manage daily habits
* Track current and best streaks
* View progress through an analytics dashboard
* Milestone-based progression system for long-term consistency
* Responsive and modern UI

---

## 🛠️ Tech Stack

* **Frontend:** React.js, Tailwind CSS
* **Backend:** Spring Boot, Spring Security
* **Database:** PostgreSQL
* **Deployment:** Vercel

---

## ⚙️ Getting Started

```bash
git clone https://github.com/sschandra723/habit-tracker.git
cd habit-tracker
```

### Backend

```bash
cd backend
mvn spring-boot:run
```

### Frontend

```bash
cd frontend
npm install
npm start
```
